# <a name="azure-data-science-virtual-machine-release-notes"></a>Azure 資料科學虛擬機器版本資訊

在本文中，您將瞭解 Azure 資料科學虛擬機器版本。 如需所包含工具的完整清單以及版本號碼，請參閱[此頁面](./tools-included.md)。

請參閱[已知問題的清單](reference-known-issues.md)，以瞭解已知的錯誤和因應措施。

> [!IMPORTANT]
> [Microsoft 身分識別平臺（v2.0）](/azure/active-directory/develop/v2-overview)是 Azure Active Directory （Azure AD）開發人員平臺（1.0 版）的演進。 它可讓開發人員建置應用程式以登入所有 Microsoft 身分識別，並取得權杖以呼叫 Microsoft Graph 等 Microsoft API，或開發人員所建置的 API。
> 此內容適用于較舊的 Azure AD v1.0 端點。 針對新的專案，建議您使用 v2.0 端點。 如需詳細資訊，請參閱[為何要更新 Microsoft 身分識別平臺（v2.0）？](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) 以及 Microsoft 身分[識別平臺限制](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations)。
